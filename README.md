    [기능설명]
    
            난수 생성 알고리즘(LCG)을 이용한 난수 생성기


    [사용방법]
    
            1. 난수 생성 범위(Min Number, Max Number)를 입력한다.
            2. 난수 생성 횟수(Iterations)를 입력한다.
            3. Generate 버튼을 클릭해서 난수를 생성한다.
